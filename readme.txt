1.  Introduction 
______________________________________________________________________________________

Cowabunga is a level editor for the NES game, Teenage Mutant Ninja Turtles. Cowabunga 
currently has the following features:

- Editing for most of the levels in the game. (Technodrome boss is only uneditable 
  level)
- Basic Palette Editing
- TSA Editing
- Enemy Editing/Adding/Deletion
- Special Objects Editing (adding and deleting not implemented yet)
- Solidity Settings Editing
- Editing of various level properties, such as pattern tables used, changes in enemy 
  types, and a few other settings.
